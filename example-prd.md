# 在线图书管理系统 需求规格说明书

## 1. 项目背景
本系统旨在为中小型图书馆提供一个在线图书管理平台，支持图书的编目、借阅、归还、预约等功能。

## 2. 功能需求

### 2.1 用户管理
系统需要支持三种角色：普通读者、图书管理员、系统管理员。
- 读者可以注册账户、登录、修改个人信息、查看借阅历史
- 图书管理员可以添加/修改/删除图书信息、处理借阅和归还
- 系统管理员可以管理用户账户、设置系统参数

### 2.2 图书管理
- 图书信息包括：ISBN、书名、作者、出版社、出版年份、分类、库存数量、可借数量
- 支持按书名、作者、ISBN、分类进行搜索
- 图书可以有多本副本，每本副本有唯一条形码

### 2.3 借阅管理
- 读者每次最多借阅5本书
- 借阅期限为30天，可续借一次（延长15天）
- 逾期归还按每天0.5元计费
- 读者可以预约已被借出的图书
- 预约图书归还后保留48小时

### 2.4 通知系统
- 借阅成功/归还成功发送通知
- 到期前3天发送提醒
- 逾期后每天发送提醒
- 预约到书发送通知

## 3. 非功能需求
- 系统需支持至少1000个并发用户
- 页面响应时间不超过3秒
- 数据需每日自动备份
- 系统需记录所有操作日志
